# Catan Map Generator

A Catan map generator.

## Catan Games and Expansions Supported
- Base Game
- Cities and Knights
- Seafarers

### What the emojis mean:
🌵 = Desert 
🌊 = Sea 
🌾 = Field 
🧱 = Hills 
🗻 = Ore 
🌳 = Wood 
🐑 = Pastures
